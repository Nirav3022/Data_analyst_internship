# Telecom Customer Churn Analysis

## Objective
Analyze telecom customer churn using Power BI.

## Dataset
Telco customer churn dataset.

## Tasks Completed
- Data import
- Data cleaning
- Dashboard creation
- Churn analysis
- KPI reporting

## Key Insights
- Month-to-month contracts show higher churn
- Customers with high monthly charges churn more
- Longer tenure reduces churn probability

## Tools Used
- Power BI
- GitHub

## Data Transformation Steps

- Removed blank rows
- Fixed missing values
- Changed data types
- Removed duplicates
- Created Tenure Group column
- Created Charge Category column
- Cleaned text formatting
